Everything the app does

A complete intake loop — capture, respond, watch, follow up — running on its own once you connect it.

Capture

Lead capture & embed snippet

Send leads to a single intake endpoint or drop the embed snippet on any site. Every inbound enquiry lands in one place, structured and timestamped.

Respond

Instant AI first-response

The moment a lead arrives, the AI acknowledges it in seconds — specific to the enquiry, not a generic auto-reply — and sets expectations for the human follow-up.

Watch

Live Intake Monitor dashboard

See every lead as it lands, which were answered fast, and which are leaking — the same diagnostic view shown above, with your real traffic.

Recover

Follow-up automation

A follow-up cron keeps unworked leads alive on a schedule you control, so high-consideration buyers get the nudges they need to book.

Control

Settings: API key, SLA, follow-up rules

Tune response targets, set follow-up cadence, and generate an API key to wire IXIA into the tools you already run.

Billing

Self-serve billing

Upgrade, downgrade, or cancel from inside the app. Start on a 14-day free trial with no card, then move to Starter or Pro when you are ready.
